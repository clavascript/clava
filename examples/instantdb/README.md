# InstantDB

See [InstantDB](https://www.instantdb.com) for more info about InstantDB.

## Dev

Fill in your InstantDB App ID in `.env` like:

```
VITE_APP_ID=...
```

Run `npm install` and then `npm run dev` to start vite and squint.
